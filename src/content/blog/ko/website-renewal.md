## 공식 웹사이트를 리뉴얼했습니다

Acecore는 공식 웹사이트를 리뉴얼했습니다. 이 글에서는 리뉴얼의 목적, 새로운 디자인 컨셉, 그리고 기반 기술에 대해 보고합니다.

## 리뉴얼의 목적

이전 사이트는 일반적인 CMS로 구축되어 있었습니다. 기능적으로는 문제가 없었지만, 페이지 속도, SEO 최적화, 디자인 유연성 측면에서 과제가 있었습니다.

이번 리뉴얼에서는 다음과 같은 목표를 설정했습니다:

1. **퍼포먼스**: PageSpeed Mobile 95+ 달성
2. **SEO**: 구조화 데이터 및 OGP의 적절한 구현
3. **접근성**: WCAG AA 준수를 목표
4. **유지보수성**: 엔지니어가 직접 콘텐츠를 편집할 수 있는 기술 스택 선택

## 사용 기술

| 기술 | 용도 |
|------|------|
| Astro | 정적 사이트 생성기 |
| UnoCSS | CSS 프레임워크 |
| Cloudflare Pages | 호스팅 |
| Pagefind | 전문 검색 |
| ssgform.com | 문의 폼 |

Astro는 JavaScript가 없는 HTML을 생성하여 가장 빠른 초기 렌더링을 제공합니다. UnoCSS는 실제로 사용된 클래스의 CSS만 생성하여 전체 CSS 크기를 최소한으로 유지합니다.

## 새로운 디자인 컨셉

"심플하고, 빠르고, 신뢰할 수 있는" 컨셉을 채택했습니다.

- **심플**: 산만한 디자인을 최소화한 깔끔한 레이아웃
- **빠름**: 방문자가 필요한 정보를 즉시 찾을 수 있도록 페이지 속도 극대화
- **신뢰**: 비즈니스로서의 신뢰감을 전달하는 프로페셔널한 외관

색상 구성은 네이비 기반의 브랜드 컬러(#002855)에 화이트와 그레이를 조합하여 차분하고 높은 신뢰감을 주는 인상을 연출합니다.

## 개선 사항

### 퍼포먼스

Astro로 구축함으로써 처음부터 PageSpeed Mobile 95+ / Desktop 100을 달성했습니다. 이미지 최적화와 CSS 외부화를 구현하여 점수를 더욱 향상시켰습니다.

### SEO

JSON-LD 구조화 데이터(Organization, BlogPosting, BreadcrumbList, FAQPage), OGP, canonical 태그, 사이트맵을 적절하게 구현했습니다.

### 블로그 기능

기술 기사를 게시할 수 있는 블로그 섹션을 추가했습니다. Pagefind를 활용한 전문 검색 기능, 태그별 및 월별 아카이브 탐색 기능을 갖추고 있습니다.

### 문의 폼

ssgform.com을 사용하여 서버가 필요 없는 안전한 문의 폼을 구현했습니다. Cloudflare Turnstile을 통한 봇 방지 기능도 결합했습니다.

## 앞으로의 전망

이번 리뉴얼로 속도, SEO, 접근성을 겸비한 기반을 구축했습니다. 앞으로도 블로그와 서비스 콘텐츠를 지속적으로 강화해 나가겠습니다.
