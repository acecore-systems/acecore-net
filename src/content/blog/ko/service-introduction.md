## Acecore의 IT 서비스 소개

Acecore는 **시스템 개발, 웹 제작, IT 교육**의 세 가지 축을 중심으로 종합적인 IT 서비스를 제공합니다. 이 글에서는 각 서비스의 특징과 고객에게 제공하는 가치를 소개합니다.

## 시스템 개발

요구사항 정의부터 설계, 구현, 테스트, 배포까지 시스템 개발의 전 과정을 담당합니다.

- **웹 애플리케이션**: React, Vue.js, Astro 등을 활용한 SPA/SSR
- **API 개발**: RESTful API / GraphQL 설계 및 구현
- **클라우드 인프라**: AWS, GCP, Cloudflare를 활용한 아키텍처 설계 및 운용

기술은 각 프로젝트에 가장 적합한 것을 선정합니다. 유지보수성과 확장성도 고려합니다. "선호하는 기술"은 없습니다 — 프로젝트의 요구사항에 따라 가장 적절한 도구를 선택합니다.

## 웹 제작

"존재하는 웹사이트"와 "성과를 내는 웹사이트"는 큰 차이가 있습니다. 저희는 실제로 결과를 만들어내는 사이트를 구축합니다.

- **퍼포먼스**: PageSpeed Mobile 99+ 가 출발선
- **SEO**: 구조화 데이터, OGP, 사이트맵, RSS를 기본 탑재
- **접근성**: WCAG AA 준수를 목표
- **보안**: CSP, HSTS 등 보안 대책을 기본 적용

정적 사이트 생성기 Astro를 주요 도구로 사용하여 속도와 SEO를 동시에 달성합니다. 지금 읽고 계신 이 사이트 자체가 저희의 쇼케이스입니다.

## IT 교육

Acecore Schools에서는 현직 엔지니어가 1대1로 지도합니다. 학생들은 실제 프로젝트를 수행하며 실무 능력을 개발합니다.

- **프로그래밍**: 입문부터 실용 단계까지
- **인프라**: 서버 구축, 클라우드 관리, Linux
- **학습 지원**: 수학, 영어, 고등학교 졸업 검정고시 대비

저희의 차별점은 커리큘럼의 유연성입니다. 각 학생의 목표와 현재 수준에 맞춰 학습 경로를 커스터마이즈하여 목표 달성까지의 최단 경로를 제공합니다.

## 인프라 운용 관리

서버 구축부터 운용, 모니터링까지 포괄적으로 지원합니다.

- **서버 구축**: VPS/클라우드 서버 설정 및 최적화
- **모니터링 및 알림**: 가동 시간 모니터링, 로그 분석, 인시던트 대응
- **보안**: 방화벽 설정, 인증서 관리, 취약점 대응

저희의 운용은 Aceserver 커뮤니티에서 얻은 실무 경험을 기반으로 합니다. 고부하 환경에서의 퍼포먼스 튜닝과 DDoS 방어는 특히 자신 있는 분야입니다.
